# Proyecto de Data Science

Estructura típica de un proyecto de ciencia de datos.

## Carpetas principales
- `data/raw/`: Datos originales sin procesar
- `data/processed/`: Datos procesados/listos para análisis
- `notebooks/`: Jupyter notebooks
- `src/`: Código fuente del proyecto
- `models/`: Modelos entrenados y scripts relacionados
- `reports/`: Reportes y resultados
- `tests/`: Pruebas del código
- `figures/`: Figuras y visualizaciones
- `references/`: Documentación y referencias
- `docs/`: Documentación del proyecto
